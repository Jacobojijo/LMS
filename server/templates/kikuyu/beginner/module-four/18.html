4.12.2 Relative Pronouns and Causal Connectives

I. Introduction to Relative Pronouns in Gĩkũyũ

Relative pronouns in Gĩkũyũ function to link a principal clause to a subordinate clause, allowing complex sentence construction.

The core relative markers are the Noun Class prefixes + -rĩa (e.g., ũrĩa, kĩrĩa, ĩrĩa) — equivalent to English "that" or "which".

II. Key Relative Pronoun: The -rĩa System 🧩

1. Function and Usage

  • The core relative pronoun is used to identify or modify the subject or object of a sentence (equivalent to English "that," "which," or "who").
  • Unlike English, the Kikuyu relative pronoun is formed by taking the noun's class prefix and adding the suffix -rĩa.
  • The verb kũrĩa (to eat) is NOT the relative pronoun.

2. Contextual Examples of the Relative Marker

The correct relative particle must agree with the noun class of the noun it refers to:

Noun Class Prefix + -rĩa Gĩkũyũ Example English Translation
Class 1 (Mũ-) ũrĩa Mũndũ ũrĩa mũkũrũ The person who is old
Class 3 (Mũ-) ũrĩa Mũtĩ ũrĩa mũraĩhu The tree which is tall
Class 9 (N-) ĩrĩa Nyũmba ĩrĩa njerũ The house which is new
Class 7 (Kĩ-) kĩrĩa Kĩrĩma kĩrĩa kĩnene The hill which is big

Contextual Distinction:

The -rĩa System = "that/who/which" (links main noun to a modifying descriptive clause).

Nĩguo = "so that / in order that" (links main clause to a purpose clause, as detailed in Section III).

III. Structural Scheme: Principal + Purpose Clause Pattern

This pattern uses Nĩguo to link a main action to its intended purpose.

Principal Phrase Purpose Connector Subordinate Clause / Predicate Translation (Purpose)
Nĩnjaria nĩguo aheo maĩ. I speak so that he may be given water.
Tũraaka nĩguo nyũmba ĩrĩe. We build so that the house may last/be permanent.
Wamũrutana nĩguo mwana athome. She taught so that the child may see/learn.
Nĩndeng'eraga nĩguo njone hinya. I run so that I may gain strength.

Note: The phrase Nĩ kũrĩ ũkoragwo translates to "It is so that you may exist/be present" (directive sense).

IV. Contextual Examples and Syntax

A. Expressing Intent or Purpose (Using Nĩguo)

  • Tũhande mbembe nĩguo mwana athome. — We plant maize so that the child may see/live.
  • Njũragie gĩthaka nĩguo njone ng'ombe. — I clear the farm so that I may get cows.

B. Imperative Constructs (Using Nĩ kũrĩ)

  • Nĩ kũrĩ ũkoragwo! — You must be present (directive)!
  • Nĩ kũrĩ ũnyitire! — Do it so that you may succeed!

V. Expressing Cause: Causal Connectives in Gĩkũyũ

1. "Because" Equivalents

Used to express direct cause or reason. The most common forms are Nĩũndũ wa and Tondũ.

Kikuyu Expression English Translation
Nĩnjũragia nĩũndũ wa gũkĩra hinya. I run because I am becoming strong.
Nĩndĩaga tondũ wa ng'aragu. I eat because I am hungry.
Aathiire mũgũnda nĩũndũ wa wĩra. He went to the farm because of work/duty.
Nĩnjĩkaga ũteithio nĩũndũ wa wendo. I help because of love/kindness.

2. Other Related Causal Markers

  • Nĩgĩtũmi — because / for the reason (literally: it is the reason).

VI. Comparative Overview of Purpose (Nĩguo) and Cause (Nĩũndũ wa)

Connector Function Correct Example (Gĩkũyũ) Translation
Nĩguo Introduces clause expressing purpose Tũhande nĩguo mwana one. We plant so that the child may see/live.
Nĩũndũ wa Expresses cause / reason Nĩndĩaga nĩũndũ wa ng'aragu. I eat because I am hungry.

VIII. Summary of Key Takeaways

  • Nĩguo = "so that/in order that" (links principal clause to dependent clause expressing purpose).
  • Nĩ kũrĩ carries directive or motivational force in imperatives.
  • Nĩũndũ wa / Tondũ = "because / for the reason that", used to express cause or justification.
  • Understanding the future-oriented (purpose) vs cause-oriented (reason) distinction is crucial.
