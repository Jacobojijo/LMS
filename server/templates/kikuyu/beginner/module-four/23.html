4.15.3 Conditional Mood

I. Definition and Core Structure

The Conditional Mood in Kikuyu is used to express cause-and-effect, dependency, or hypothetical situations. It functions like the English "If… then…" construction.

Kikuyu equivalents:

  • Ningĩ = If (introduces the condition)
  • No nginya / No ũndũ wa = Then (optional, adds emphasis)

II. Sentence Structure and Tense Alignment

In Kikuyu conditional clauses:

  • Condition Clause (Ningĩ…) → Usually in Present Tense
  • Result Clause → Usually in Future Tense using the {-gũ-} marker

Pattern:

Ningĩ + Present Tense Verb + (No nginya) + Future Tense Verb

III. Core Examples and Breakdown

  1. Ningĩ wũka, no nginya ngũone. → If you come, then I will see you.
  2. Ningĩ arĩa, no nginya ndĩmũhe mũtĩ. → If he eats, then I will give him a tree (example action).
  3. Ningĩ ũhote kũhoya, no nginya ngũhe ũhoro. → If you can ask, I will give you the information.
  4. Ningĩ maakinya, no nginya ndĩmageithie. → If they arrive, I will greet them for us.
  5. Ningĩ aina, no nginya tũcũngage. → If he/she sings, then we will dance.

Observation: "No nginya" is optional but strengthens the consequence.

IV. Negative Conditional Structures

Negation is applied to both the condition and the result.

Example:

Ningĩ waga kũcooka, no nginya ndingĩkuona. → If you do not return, then I will not see you.

Structure:

Ningĩ + negative present verb + (no nginya) + negative future verb

V. Composed Past Conditionals

Used for events that did not happen, but could have happened under different circumstances.

Kikuyu uses:

  • "Ningĩ nĩathire atĩ…" = If it had been that…
  • "Ningĩ ndĩ…(past)" = introduces the hypothetical past

Examples:

  1. Ningĩ ndĩmenyete atĩ mũciĩ mũgũthoma, ningĩ ndathire hamwe. → If I had known the meeting was starting, I would have gone.
  2. Ningĩ nĩamenyete atĩ ndakũũga, ningĩ arutire gĩthomo kĩa nginya. → If she knew he wouldn't call, she would have continued studying.

Insight: These structures express counterfactuals — things that did not happen.

VI. Remote Past Conditionals

Used in reflection, storytelling, and moral lessons.

Examples:

  1. Ningĩ ndĩmenyetie mũndũ ũcio, ningĩ ndatũmire ũhoro wake. → If I had known that person, I would not have trusted his words.
  2. Ningĩ nĩamenyete atĩ ti mũndũ wa wega, ningĩ ndagire thirikari. → If she had known he wasn't a good man, she wouldn't have agreed to the marriage.

These forms add depth to narrative expression.

VII. Future Conditional Forms

Used for planning, predictions, and promises.

Examples:

  1. Ningĩ obiro gũka rĩu (common Nyeri form: "ningĩ ũka rũciũ"), no nginya ningũkuhe. → If he will come, I will give him.
  2. Ningĩ ũmũhe ũhoro wakwa, no nginya ngũũka. → If you give him my message, he will come.

Observation: The future marker {-gũ-} can appear in both clauses.

VIII. Summary of Key Patterns

Structure Type Kikuyu Pattern English Equivalent
Basic Conditional Ningĩ + Present, (no nginya) + Future If + present, (then) will + verb
Negative Conditional Ningĩ + negative present, (no nginya) + negative future If not…, (then) will not…
Past Conditional Ningĩ ndĩ…/Ningĩ nĩathire atĩ… If (I/he) had…, (I/he) would have…
Remote Hypothetical Same structure, used for deep reflection Retrospective "what if…"
Future Conditional Ningĩ + future verb, (no nginya) + future If he will…, he will…

IX. Concluding Insights

The Ningĩ… no nginya… pattern in Kikuyu:

  • Links present situations to future outcomes
  • Allows speakers to express hypothetical situations
  • Strengthens reasoning, argumentation, planning, and narrative skill
  • Is essential for formal, academic, and cultural discourse

Mastery of conditional mood builds fluency in expressing decisions, consequences, moral lessons, and reflective thought.
