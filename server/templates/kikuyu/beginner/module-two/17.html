2.5.5 Distributive Pronouns

Overview — what distributive pronouns do

Definition: Distributive pronouns refer to individual members or items within a group, stressing each separately rather than the group as a whole.

English equivalents: each, every, either, neither.

Functions: (a) quantify over individuals, (b) single out items for separate treatment, (c) create distributive statements (e.g., "each person did X"), and (d) form indefinite reference (any, anything).

1. "Each one" — persons (Mũndũ kwa mũndũ)

Literal gloss: person by person / one person at a time.

Structure: Mũndũ (person) + kwa mũndũ (each individual person).

Use: To speak of every individual in a group separately — distributive over people.

Key morphological pieces:

  • Mũndũ = person
  • kwa mũndũ = each (individual) person

Canonical examples (from source):

Mũndũ kwa mũndũ atũhikanire. — Each one is married.

Mũndũ kwa mũndũ ũrĩa athiĩ kĩa gĩkũyũ. — Each person who speaks Kikuyu.

Practical note: Use this structure when you mean every single person, emphasising individuality.

2. "Each one" — things (kĩndũ / ciũndũ patterns)

Noun-class background (as given): the thing/item class uses the k~ prefix; singular kĩndũ, plural ciũndũ.

Preferred distributive phrasing (things):

  • kĩndũ kĩmwe na kĩmwe — literally "one thing and one thing" → each item / each one (thing).
  • o kĩndũ — alternative structure meaning each thing / every thing (intensive).

Why these forms: they explicitly iterate items (kĩmwe = one), producing a grammatically robust distributive reading.

Examples (from source):

Kĩndũ kĩmwe na kĩmwe gĩtĩkĩĩtie nĩ kĩhũthagĩra. → Each item has a purpose.

O kĩndũ gĩtĩkĩĩtie nĩ gĩtũũra. → Every thing has value.

Practical note: For distributive reference to things, prefer the kĩndũ kĩmwe na kĩmwe pattern or the o kĩndũ alternative for emphasis.

3. "Any one" — persons (mũndũ ũmwe wothe / mũndũ o wothe)

Meaning: a non-specific / random individual — English anyone / any one.

Attested forms (both grammatical in the data):

  • mũndũ ũmwe wothe (literally person one all/any)
  • mũndũ o wothe (more compact: even any person; intensifier o included) — noted as more common / simpler in the data.

Component glosses (from source):

  • mũndũ = person
  • ũmwe = one
  • wothe = any / all
  • o = intensifier (even / every)

Examples (from source):

Mũndũ o wothe no athikĩrĩrie. — Anyone can listen.

Mũndũ o wothe no oke. — Anyone can come.

Mũndũ o wothe no arie. — Any person may speak.

Practical note: Use mũndũ o wothe for generic, indefinite reference to persons.

4. "Any one" — things (kĩndũ o kĩothe / kĩndũ kĩmwe o kĩothe)

Meaning: non-specific item — English anything / any one thing.

Attested forms (from source):

  • kĩndũ o kĩothe — even any thing (common, grammatically sound)
  • kĩndũ kĩmwe o kĩothe — alternate explicit form with kĩmwe = one.

Component glosses (from source):

  • kĩndũ = thing / item
  • kĩmwe = one
  • o = intensifier (even / every)
  • kĩothe = any / all (for the k~ noun class)

Examples (from source):

Twĩre kĩndũ gĩothe kĩna ũru. — Tell us any one thing that is bad.

Thiĩ ũcore kĩndũ kĩmwe o gĩothe. — Go and draw any one thing.

Hoya kĩndũ o kĩothe. — Ask for anything.

Practical note: Use kĩndũ o kĩothe for broad, indefinite requests or permissions referring to objects.

5. "All" / "Absolutely all" (-othe with class prefix)

Core device: the suffix -othe combined with the appropriate noun-class prefix expresses totality (all / absolutely all).

Generalisation in the data: wothe is the form for the mũndũ (person) class; other classes vary their prefix.

Attested noun-class table (from source):

Noun class (prefix) Singular form Plural form Meaning
Mũ-/A- wothe othe all (people)
Mũ-/Mĩ- wothe yothe all (trees, things)
Kĩ-/Ci- kĩothe ciothe all (things, items)
N-/N- yothe ciothe all (animals, nouns)

Attested example sentences

Mĩtĩ yothe nĩ mĩthaka. — All the trees are beautiful.

Andũ othe nĩmegũka. — All the people will come.

Ciũngũ ciothe nĩ thaka. — All the pots are good.

Tũgũcaria maĩ mothe. — We will look for all the water.

Mĩgũnda yothe nĩ yaku. — All the farms are yours.

Practical note: Attach -othe with the correct class prefix to express totality; context gives emphasis (variants such as -ũyo were noted as possible intensifiers but not standard for "absolutely all" in the provided material).

6. "Anything" (kĩndũ kĩothe / kĩndũ o kĩothe)

Canonical expression (from source): kĩndũ kĩothe — literally any thing.

Intensive alternative: kĩndũ o kĩothe — anything at all.

Component glosses: kĩndũ (thing) + kĩothe (any).

Examples (from source):

Nĩngũgũtũma kĩndũ kĩothe kĩrĩa ũkĩrũĩte. — I will send you anything you want.

Nĩũgũtũma kĩndũ kĩothe? — Are you sending anything?

Twĩĩthe kĩndũ kĩothe gĩa gũthũkũma. — We own anything for work.

Practical note: kĩndũ kĩothe is the most accurate and general form for anything.

7. "Something" (kĩndũ / kĩndũ kĩna)

Basic word: kĩndũ — a thing / something. In many contexts this alone conveys something.

To specify "a certain / particular thing": kĩndũ kĩna — a certain thing / a particular thing (kĩna = a certain / particular).

Examples (from source):

Nĩngũhoroirie kĩndũ. — I heard something.

Kĩndũ kĩmũtũmaga kĩũrũru. — Something is making him sad.

Kĩndũ kĩna nĩkĩrĩĩte gwakwa. — Something is missing from my house.

Practical note: Use kĩndũ for unspecific mention; add kĩna when you mean a particular/specific item.
