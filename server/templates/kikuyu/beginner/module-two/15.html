2.5.3 Possessive Pronouns and Cases

1. Overview — core facts

Two main strategies for expressing possession in Kikuyu:

  • Possessive adjectives / pronoun-suffixes that agree with the noun class (suffix attached to the noun or to a class prefix).
  • Connector + noun/pronoun using "wa" (rendered in class-adjusted shapes in examples) meaning of / belonging to.

The form of the possessive depends on three factors:

  • Person (1st / 2nd / 3rd),
  • Number (singular / plural), and
  • Noun class (possessives show class agreement in prefix shape).

Two surface devices are therefore available: short suffix forms (e.g., -akwa) and wa + pronoun / wa + noun constructions (e.g., wa niĩ, Mwana wa Mũthoni).

2. Singular possessive forms — table + notes

Forms provided in the source (singular):

Person Short/suffix form Alternative 'wa' form Examples
1st sing. (I / me) -akwa wa niĩ Mwana wakwa = "my child" ; Nyumba ya niĩ = "my house"
2nd sing. (you) -aku wa wĩ Mwana waku = "your child" ; Gĩthaka gĩaku = "your garden"
3rd sing. (he/she/it) -ake (no explicit wa+pronoun example given for 3rd sing.) Mwana wake = "his/her child" ; Nyumba yake = "his/her house"

Observations

  • The short suffixes (-akwa, -aku, -ake) are the primary short possessive forms shown.
  • The wa + pronoun construction is supplied as an alternative or emphatic form for 1st and 2nd singular (e.g., wa niĩ, wa wĩ).
  • The examples show class-adjusted surface shapes (e.g., Nyumba ya niĩ, Gĩthaka gĩaku) — i.e. the possessive expression appears with a prefix that matches the noun's class in the examples.

3. Plural possessive forms — table + notes

Forms (plural):

Person Short/suffix form Alternative / wa-form Examples
1st plural (we / us) -itu (explicit earlier in text) — Mwana witu = "our child" ; Gĩthaka gitu = "our garden"
2nd plural (you all) -anyu — Mwana wanyu ; Nyumba yanyu = "your (pl.) house"
3rd plural (they / them) -ao or wa acio (both forms recorded) — Mwana wao = "their child" ; Gĩthaka kiao = "their farm"

4. The connector "wa" — how it is used (point form + examples)

  • Core meaning: wa functions like English "of / belonging to".
  • Construction: Wa + noun/pronoun shows clear ownership (example: Mwana wa Mũthoni = "Mũthoni's child").
  • Class agreement in examples: the examples provided show the connector surfacing with class-adjusted shapes in context (e.g., Nyumba ya niĩ — ya surfaces with nyumba; Gĩthaka kiao / kĩao appears with gĩthaka).

Representative examples

  • Mwana wa Mũthoni — "Mũthoni's child"
  • Nyumba ya niĩ — "my house"
  • Gĩthaka kiao — "their farm" (one attestation)
  • Gĩthaka kĩao — "the garden is theirs" (another attestation in the real-life sentence set)

6. Important notes

  • Gender neutrality: the 3rd person singular possessive -ake is used for both male and female ("his/her").
  • Agreement: possessive pronouns often change their prefix to agree with the noun class (explicit example: nyumba yake, gĩthaka gĩake).
  • Dual options for expression: you can use the short possessive suffix (e.g., wakwa) or the wa + pronoun style (e.g., wa niĩ) depending on emphasis and syntactic choice;
