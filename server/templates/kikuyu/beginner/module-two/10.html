2.3.4 Noun Plurals

Structure of the System

Plural formation in Kikuyu is a highly morphological process grounded in prefix alternations, invariance, and occasional irregularity. The system reflects both semantic categorization (e.g., people, animals, tools, plants) and morphophonological rules (prefix substitution, vowel assimilation, or stability).

The pluralization system can be explained under six major classes:

General Structural Principles

Prefix Alternation:

Plurality is most often marked through prefix substitution, where singular prefixes are systematically replaced by plural prefixes.

Example: mũ- → a- (human nouns).

Invariance:

Some nouns do not morphologically change between singular and plural, requiring contextual interpretation.

Irregularity:

A subset of nouns undergo lexical replacement or irregular changes, producing plurals with little or no formal similarity to the singular.

Semantic Motivation:

Different noun classes correlate with semantic domains such as humans, tools, plants, animals, kinship terms, or natural elements.

Class I – mũ- → a- Class (Human Class)

Pattern:

  • Singular prefix: mũ-
  • Plural prefix: a-
  • Rule: Replace mũ- with a- when forming plurals.

Semantic Domain:

  • Used for human beings (persons, professions, kinship roles).
  • Also extended to a few animals that share anthropomorphic qualities or social significance.

Examples:

  • mũndũ → andũ (person → people)
  • mũtumia → atumia (woman → women)
  • mũthoni → athoni (in-law → in-laws)
  • mũhiki → ahiki (bride → brides)
  • mũhunjia → ahunjia (pastor → pastors)

Key Feature:

Central class for social relations—it encodes collective human identity.

Class II – gĩ-/kĩ- → i- Class (Object Class)

Pattern:

  • Singular prefixes: gĩ- / kĩ-
  • Plural prefix: i-
  • Rule: Replace gĩ-/kĩ- with i-.

Semantic Domain:

  • Refers to tools, objects, plants, and small animals.
  • Marks plurality of utilitarian and functional items.

Examples:

  • muti → miti (tree → trees)
  • kĩnde → inde (rope → ropes)
  • kĩega → njega (good thing → good things)
  • mũgũnda → mĩgũnda (farm → farms)
  • kĩhingo → ihingo (gate → gates)

Key Feature:

  • Central to the material and agricultural world.
  • Often marks countable objects used in daily life.

Class III – iri- / rĩ- → ma- Class (Body Parts & Fruits Class)

Pattern:

  • Singular prefixes: iri- / rĩ-
  • Plural prefix: ma-
  • Rule: Replace rĩ- with ma-.

Semantic Domain:

  • Used for body parts, fruits, and certain objects.
  • This reflects items that exist in natural pairs or multiples.

Examples:

  • itumbe → matumbe (egg → eggs)
  • gũtũ → matũ (ear → ears)
  • igego → magego (tooth → teeth)
  • igũrũ → magũrũ (leg → legs)

Key Feature:

Encodes natural multiplicity (parts that normally occur in sets).

Class IV – n-/m- Invariant Class (Stable Class)

Pattern:

  • Singular prefix: n-/m-
  • Plural prefix: same as singular
  • Rule: Singular = Plural (no morphological change).

Semantic Domain:

Covers animals, borrowed words, and natural phenomena.

Examples:

  • mbũri → mbũri (goat → goats)
  • ng'ombe → ng'ombe (cow → cows)
  • mbura → mbura (rain → rains)
  • ndĩri → ndĩri (drum → drums)
  • mbeũ → mbeũ (seed → seeds)

Key Feature:

Contextual plurality: requires the surrounding words (numerals, quantifiers) to disambiguate.

Class V – Irregular Plurals (Kinship & Special Nouns)

Pattern:

  • Irregular—plural form changes unpredictably.
  • No strict morphological rule; plurals often lexicalized.

Semantic Domain:

Includes kinship terms, social roles, and culturally significant nouns.

Examples:

  • mũrata → arata (friend → friends)
  • mũrogi → arogi (witch → witches)
  • mũciĩ → mĩciĩ (homestead → homesteads)
  • mũthuri → athuri (man → men)

Key Feature:

Must be memorized individually, since no universal rule applies.

Class VI – mũ- / mu- → mi- Class (Plants & Long Objects Class)

Pattern:

  • Singular prefix: mũ- / mu-
  • Plural prefix: mi-
  • Rule: Replace mũ- with mi- for long or plant-based items.

Semantic Domain:

Trees, plants, long objects, and natural elements.

Examples:

  • mũtĩ → miti (tree → trees)
  • mũhũko → mihũko (branch → branches)
  • ihiga → mahiga (stone → stones)

Key Feature:

Marks plurality in the natural environment, especially vegetation and elongated objects.

Summary Table of Kikuyu Noun Plural Classes

Creative Mnemonic Anchors

  • Class I (mũ- → a-) → "Man becomes All" → Humans multiply into communities.
  • Class II (kĩ-/gĩ- → i-) → "Kit becomes Itemized" → Objects and tools gather in sets.
  • Class III (rĩ- → ma-) → "Rĩpart becomes Many" → Body parts and fruits exist in multiples.
  • Class IV (n-/m- same) → "Nature Never Changes" → Animals and natural things resist change.
  • Class V (Irregular) → "Village Vocabulary" → Kinship and special nouns must be remembered.
  • Class VI (mũ- → mi-) → "Mũgumo → Mighty Forest" → Trees and long plants multiply as forests.
