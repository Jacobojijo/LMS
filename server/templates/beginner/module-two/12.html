2.4.1 Summary of Nouns (Part 02)

Nouns in Dholuo: Concrete & Abstract

1. Concrete Nouns

Definition: Tangible, physical objects or beings perceived via the senses.

Categories & Examples:

  • People: Japuonj (teacher), Osiep (friend), Nyathii (child)
  • Animals: Guok (dog), Nyambura (cat), Liech (elephant)
  • Places: Alap (field), Buore (city), Skul (school)
  • Things: Kom (chair), Buk (book), Nyamburko (car)
  • Materials: Yien (wood), Nyinyo (steel)

Sample Sentences:

  • Ne aneno guok buore. – "I saw the dog in the city."
  • Nobedo o kom kasto oyaro buge. – "She sat on the chair and opened her book."

2. Abstract Nouns

Definition: Intangible concepts, emotions, or qualities not perceived by senses.

Categories & Examples:

  • Emotions: Mor (happiness), Kuyo (sadness), Mirima (anger), Luor (fear)
  • Qualities: Ber (beauty), Muolo (humility), Teko (strength), Adiera (honesty)
  • Ideas: Hera (love), Thuolo (freedom)
  • Conditions: Ngima (health), Dhier (poverty), Kwee (peace)

Sample Sentences:

  • Morne nonenore bang' wach maber. – "His happiness was evident after the good news."
  • Ratiro long'o e gweng. – "Justice is important in every society."

Luo Noun Pluralization: Fun & Functional Grammar

General Structure

Luo nouns are grouped into classes based on suffix patterns, semantic roles, or morphological structure.

Plural formation varies by:

  • Suffix changes: e.g., -nde, -che, -mbe, -je, -ni
  • Invariable forms: Singular = Plural
  • Irregular forms: Plural completely different
  • Agent nouns: Ja- (singular) → Jo- (plural)

Class I: -nde Class

  • Suffix: -nde added to plural
  • Used for: Body parts, tools, small creatures
  • Examples:
    • Odundu → Odunde (pipe → pipes)
    • Pien → Piende (skin → children)
    • Ogwal → Ogwende (frog → frogs)
    • Tol → Tonde (rope → ropes)

Class II: -che Class

  • Suffix: -che for plural
  • Used for: Birds, animals, natural objects
  • Examples:
    • Akuru → Akuche (dove → doves)
    • Apuoyo → Apuoche (rabbit → rabbits)
    • Or → Oche (son-in-law → sons-in-law)
    • Wach → Weche (word → words)

Class III: -mbe Class

  • Suffix: -mbe marks plural
  • Used for: Living beings, natural items
  • Examples:
    • Bim → Bimbe (ape → apes)
    • Cham → Chambe (grain → grains)
    • Nam → Nambe (lake → lakes)
    • Thum → Thumbe (music → musical pieces)

Class IV: -je / -ye Class

  • Suffixes: -je and -ye
  • Used for: People, nature, abstract ideas
  • Examples:
    • Lur → Luje (barren woman → barren women)
    • Mach → Meje (fire → fires)
    • Kwan → Kweye (count → counts)
    • Dayo → Deye (grandmother → grandmothers)

Class V: Invariant (Le Class)

  • No suffix change: Singular = Plural
  • Used for: Collectives, animals, insects
  • Examples:
    • Rech → Rech (fish → fish)
    • Le → Le (animal → animals)
    • Kich → Kich (bee → bees)

Class VI: Irregular Plurals

  • Pattern: Plural is completely different from singular
  • Used for: Humans, household items, abstract nouns
  • Examples:
    • Dhako → Mon (woman → women)
    • Ot → Udi (house → houses)
    • Dala → Mier (home → homes)
    • Chogo → Choke (bone → bones)

Class VII: Ja → Jo Class (People/Professions)

  • Pattern: Prefix change from Ja- → Jo-
  • Used for: Agents, roles, professionals
  • Examples:
    • Jagam → Jogam (mediator → mediators)
    • Japuonj → Jopuonj (teacher → teachers)
    • Jakom → Jokom (chairperson → chairpersons)
    • Jajuok → Jojuogi (wizard → wizards)

Class VIII: -ni Class

  • Suffix: -ni added for plural
  • Used for: Body parts, abstract, possessions
  • Examples:
    • Adundo → Adundni (heart → hearts)
    • Anyuola → Anyuolni (offspring → offsprings)
    • Higa → Higni (year → years)
    • Mitoka → Mitokni (vehicle → vehicles)

Noun Construct (Singular & Plural Forms)

What It Is

A base noun form used to:

  • Attach demonstratives (e.g., this/that)
  • Add possessives (covered separately)

Forming the Construct (Singular)

  • Start with plural noun
  • Drop final vowel
  • puothe → puoth-
  • chiembe → chiemb-

Irregular Construct Examples

Meaning Singular Plural Construct
House ot udi od-
Woman dhako mon chi-

Plural Construct = Usually Same as Plural

  • chiembe → chiembe-
  • udi → ute- (exception)
  • mon → mond-
