2. Genitive Case ‚Äì Showing Possession

Genitive = possessive form, typically alters noun endings.
Usually involves consonant-based transformations or stress shifts.

Examples of Genitive Declension:

Noun (Root) Genitive Form Meaning
Lum (grass) Lumb of the grass
Diel (goat) Diend of the goat

3. Pronunciation Shifts (No Spelling Change)

  • Lak (tooth) ‚Äì stays Lak, but emphasis changes.
  • Kom (chair) ‚Äì accent shift to indicate possession.

4. Declension Patterns

  • Consonant Endings: Most genitive forms end in consonants.
  • Accent-based Changes: Some nouns rely only on stress/emphasis for genitive use.
  • Non-genitive Cases: No inflection ‚Äî nouns retain their base forms.

5. Key Notes for Learners

  • Genitive is key ‚Äì The only case that changes nouns.
  • Other cases = stable ‚Äì No changes across subject/object/prepositional roles.
  • Watch pronunciation! ‚Äì In many cases, intonation carries meaning.
  • Possession = suffix + context ‚Äì Genitive often paired with possessive suffixes or "mar" phrases (e.g., mar Adhiambo = Adhiambo's).

Introduction to Adjectives in Dholuo

Definition:

Adjectives describe or modify nouns/pronouns ‚Äî they add color, size, shape, quality, or emotional context to words.

Functions:

  • Describe nouns/pronouns: Give more detail.
  • Paint clearer pictures: Add vividness and specificity.

Examples:

  • Red apple ‚Üí Describes color
  • Tall building ‚Üí Describes height
  • Broken window ‚Üí Describes state

Types of Adjectives

  • Descriptive ‚Äì big, blue, old, happy
  • Comparative ‚Äì taller, smarter, better
  • Superlative ‚Äì tallest, smartest, best
  • Demonstrative ‚Äì this, that, these, those
  • Possessive ‚Äì my, your, his, her, its, our, their
  • Interrogative ‚Äì what, which, whose
  • Proper ‚Äì French, American (from proper nouns)

Key Rule: Add a suffix directly to the noun to indicate possession.

Phonetic and Spelling Changes

Certain nouns undergo form changes in possession:

  • "dh" ‚Üí "th":
    • Puodho ‚Üí Puotha (my garden)
  • Root changes:
    • Pala (knife) ‚Üí Pand-a (my knife)
    • Ot (house) ‚Üí Od-a (my house)

Common Noun-to-Possessive Transformations

Regular Examples:

  • Chik (law) ‚Üí Chik-na, Chik-ni, Chik-ne, etc.
