2.3.4 Noun Plurals

Structure of the System

This system comprises several major noun classes, each marked by specific pluralization patterns, including:

  • Suffix changes (e.g., -nde, -che, -mbe, -je, -ni),
  • Invariable forms where singular and plural are identical,
  • Irregular forms where plural nouns are completely different from their singular counterparts,
  • Human/Agent nouns prefixed with Ja- in singular and Jo- in plural.

Class I – -nde Class

Pattern: Singular nouns end in various forms; plurals formed by adding -nde.

Used For: Mostly body parts, natural elements, some animals.

Examples:

  • Odundu → Odunde (pipe → pipes)
  • Duol → Duonde (voice → skins)
  • Pien → Piende (skin → children)
  • Ogwal → Ogwende (frog → frogs)
  • Pino → Pinde (wasp → wasps)
  • Rabuon → Rabuonde (Potato → Potatoes)
  • Ten → Tende (string → strings)
  • Thuol → Thuonde (snake → snakes)
  • Tol → Tonde (Rope → Ropes)

Class II – -che Class

Pattern: Plurals formed with -che ending.

Used For: Animals, birds, insects, natural objects.

Examples:

  • Akuru → Akuche (dove → doves)
  • Aluru → Aluche (quail → quails)
  • Apuoyo → Apuoche (rabbit → rabbits)
  • Bor → Boche (cloud → clouds)
  • Bura → Buche (meeting → meetings)
  • Dwe → Dweche (moon → moons)
  • Maro → Mache (mother-in law → mothers-in laws)
  • Ndara → Ndeche (road → roads)
  • Oguyo → Oguche (butterfly → butterflies)
  • Ong'er → Ong'eche (monkey → monkeys)
  • Or → Oche (Son in-law →Sons in-law)
  • Otoyo → Otoche (hyena → hyenas)
  • Picha → Piche (picture → pictures)
  • Wach → Weche (word → words)
  • Wuor → Wuoche (shoe → shoes)

Class III – -mbe Class

Pattern: Plural formed by -mbe suffix.

Used For: Mostly living beings and natural elements.

Examples:

  • Bim → Bimbe (ape → apes)
  • Rombo → Rombe (sheep → sheep)
  • Cham → Chambe (grain →grains)
  • Kalam → Kalembe (pen → pens)
  • Kom → Kombe (chair → chairs)
  • Libamba → Libembe (foreign neighbour → foreign neighbours)
  • Lum → Lumbe (grass → grasses)
  • Nam → Nambe (lake → lakes)
  • Olemo → Olembe (fruit → fruits)
  • Thim → Thimbe (forest → forests)
  • Thum → Thumbe (music → musical pieces)

Class IV – -je &-ye Class

Pattern: Plural formed using -je &-ye.

Used For: People, places, phenomena.

Examples:

  • Lur → Luje (barren woman → barren women)
  • Ng'ech → Ng'eje (monitor lizard → monitor lizards)
  • Mach → Meje (fire → fires)
  • Piny → Pinje (land → lands)
  • Winy → Winje (bird → birds)
  • Ngech → Nyeye (lightning → lightnings)
  • Luche (variation of "Lur") → Luje
  • Kwan → Kweye (count → counts)
  • Dayo → Deye (grandmother → grandmothers)

Class V – Invariant Plurals (Le Class)

Pattern: No change in plural form.

Used For: Collective nouns, animals, insects.

Examples:

  • Rech → Rech (fish → fish)
  • Liech → Líech (Elephant → Elephants)
  • Le → Le (animal → animals)
  • Kich → Kich (bee → bees)

Class VI – Irregular Plurals

Pattern: Complete word change in plural form.

Used For: Human beings, household items, abstract nouns.

Examples:

  • Dhako → Mon (woman → women)
  • Dichuo → Chuo (man → men)
  • Chogo → Choke (bone → bones)
  • Luth → Ludhe (rod → rods)
  • Ot → Udi (house → houses)
  • Rawo → Rewe (war → wars)
  • Dala → Mier (home → homes)

Class VII – Ja → Jo Class (People/Nationalities/Professions)

Pattern: Nouns referring to people change from "Ja-" (singular) to "Jo-" (plural).

Examples:

  • Jagam → Jogam (mediator → mediators)
  • Jariembo → Joriembo (sender → senders)
  • Jathieth → Jothieth (healer → healers)
  • Jakeno → Jokeno (treasurer → treasures)
  • Japuonj → Jopuonj (teacher → teachers)
  • Jagedo → Jogedo (builder → builders)
  • Jaduong' → Jodongo (elder → elders)
  • Jajuok → Jojuogi (wizard → wizards)
  • Jagoro → Jogoro (designer → designers)
  • Jakom → Jokom (chairperson → chairpersons)
  • Jajuogo → Jojuoko (tax collector → tax collectors)
  • Jakuo → Jokuo (thief → thieves)
  • Jamatho → Jometho (drunkard → drunkards)
  • Jamecho → Jomecho (thief → thieves)
  • Jamiel → Jomiel (dancer → dancers)
  • Janek → Jonek (killer → killers)
  • Jater → Joter (remarrier of widow → same plural)
  • Jayath → Joyath (doctor → doctors)

Class VIII – -ni Class

Pattern: Plurals formed with -ni.

Used For: Abstract items, body parts, possessions.

Examples:

  • Adundo → Adundni (heart → hearts)
  • Anyuola → Anyuolni (offspring → offsprings)
  • Apaka → Apekni (wave → waves)
  • Bunde → Bundni (gun → guns)
  • Dak → Degni or Degi (pot → pots)
  • Higa → Higni (year → years)
  • Kanyna → Kenyni (donkey → donkeys)
  • Law → Lewni (clothes → clothes)
  • Soko → Sokni (well → wells)
  • Mitoka → Mitokni (vehicle → vehicles)
