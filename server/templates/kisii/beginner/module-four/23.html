4.15.3 Conditional Mood

I. Definition and Core Structure

  • The Conditional Mood in Ekegusii is a grammatical structure used to express cause-and-effect relationships, dependency, or hypothetical scenarios.
  • It corresponds to English "If…then…" constructions.
  • Dholuo equivalents:
    • ONYE = "If" (antecedent)
    • RIRORIO, NANGA = "Then" (consequence; optional but sometimes used for emphasis)

II. Sentence Structure and Tense Alignment

  • Antecedent (ONYE clause): Always in Present Tense
  • Consequence (RIRORIO, NANGA clause): Always in Future Tense using the NACHA form (future auxiliary verb)

Pattern: ONYE + [Verb: Present Tense] + (RIRORIO) + [Verb: Future Tense (NACHA)]

III. Core Examples and Breakdown

  1. Onye kogocha, ngogenda tore → If you come, we will go.
  2. Onye ogokorama, ngomoaka inde → If he abuses me, I will beat him.
  3. Onye gokoruga, ngoragera tore → If you cook, we will eat.
  4. Onye bagocha, ngobatebiande. → If they come, I will tell them.
  5. Onye ogotera, ngotenga tore → If he/she sings, we will dance.
  6. Onye ogokwa, ngomanya tore → If he/she dies, we will know.

Observation: The "RIRORIO" particle is grammatically optional, but it emphasizes or sharpens the logical consequence when included.

IV. Negative Conditional Structures

Constructed by negating both the antecedent and the consequence:

Example: Onye tokong'akana, rirorio tingogokorera 'meremo → If you don't pay me, then I will not work for you.

Structure: ONYE + ok + [verb] + (RIRORIO) + ok + [future verb]

V. Composed Past Conditionals

  • Used to describe what would have happened under a hypothetical condition that did not occur.
  • Formed using "yabeire boigo" (It had been that…)

Examples:

  1. Onye namanya ng'a ngotwa ere, nanga nabogoria riburi. → If I knew it would rain, I would have carried an umbrella.
  2. Onye namanyete ng'a tari gochia komonywoma, nanga narigetie omosacha omong'ao. → If she knew he wouldn't marry her, she would have looked for another man.

Construction Insight:

  • Use of "nanga" introduces the conditional perfect ("would have...").
  • These forms depict counterfactuals—events that did not happen.

VI. Remote Past Conditionals

  • Employed for deeply hypothetical or retrospectively reflective situations.
  • Frequently used in narratives, moral lessons, and personal introspections.

Examples:

  1. Onye namomanyete, nanga timwegeneti. → If I had known him, I would not have trusted him.
  2. Onye namanyete ng'a tari 'monto bw'ekeene, nanga ntanchete konywomu. → If she had known he was not an honest man, she would not have accepted the marriage.

VII. Future Conditional Forms

  • Utilized for projected events or anticipated dependencies.
  • Frequently seen in planning, persuasion, and promises.

Examples:

  1. Onye ere ogocha, ngomotebia inde.. → If he will come, I will tell him.
  2. Onye ogoancha okobori kwane, ngong'anchera are ing'ende. → If he accepts my request, he will let me go.

VIII. Summary of Key Patterns

Structure Type Ekegusii Pattern English Equivalent
Basic Conditional Onye + Present, (rirorio) Future (Nacha) If + present, (then) will + verb
Negative Conditional Onye tari + Present, (nanga) ta… + Future If not + present, (then) will not + verb
Past Conditional Onye na… + Past, nanga… If (I/he) had..., (I/he) would have...
Remote Hypothetical Same as above, used for deeper reflection Retrospective "what if" clauses
Future Hypothetical Onye na… + verb, nanga na… + verb If he will..., he will...

IX. Concluding Insights

  • The ONYE...NANGA construction is a logical bridge in Ekegusii syntax and discourse.
  • It connects present realities with future possibilities, or past assumptions with unrealized outcomes.
  • Mastery of this structure is essential for:
    • Argumentation
    • Planning
    • Narration
    • Emotional appeal
  • Sociolinguistic nuance in both spoken and written Ekegusii.
