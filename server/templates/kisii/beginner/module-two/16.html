2.5.5 Distributive Pronouns

Distributive pronouns in English are words like each, every, either, and neither. They refer to individuals within a group, rather than the group as a whole. This means they highlight each member being considered separately, emphasizing individual actions or characteristics.

1. "Each One" (referring to a person):

Kera omonto

  • Literally: person by person
  • Used when referring to every single individual separately
  • Structure:
    • Omonto = a person
    • Kera omonto = each individual person
  • Example:
    • Kera omonto ngochia koragera are — Each one will eat.
    • Kera omonto orosirie omogano oye — Each person has made his story.

2. "Each One" (referring to a thing):

Kera egento

  • Literally: thing by thing
  • Refers to each individual item in a group
  • Structure:
    • Egento = a thing/something
  • Example:
    • Kera oyomo nabo agokora emeremo — Each one (thing) can work.
    • Kera egento ngebwate engencho yaye — Each thing has its meaning.

3. "Any One" (referring to a person):

Onde bwensi

  • Used to refer to any one person, non-specific
  • Expresses randomness or unspecified individual
  • Structure:
    • Omonto = a person
    • onde bwensi = any one
  • Example:
    • Omonto onde bwensi nabo akoba omobui — Anyone can be a winner.
    • Omonto onde bwensi nabo akogenda isiko — Any person can go out.

4. "Any One" (referring to a thing):

Kende gionsi

  • Used to refer to any one item, random or unspecified thing
  • Structure:
    • Kende = one
    • Kende gionsi = any one thing
  • Example:
    • Beka kende gionsi ase emesa — Put any one on the table.
    • Nabo ogochora kende gionsi — You may choose any.

5. "All"

Bionsi/Bonsi

  • Refers to all things or all people, collectively
  • Bionsi/Bonsi is emphatic — "Absolutely all!"
  • Structure:
    • Bionsi = all
    • Bonsi = intensifier (adds emphasis)
  • Example:
    • Abaamate baye bonsi ngochabare — All their relatives will come.
    • Namaene endagera yonsi eyarietu — Absolutely all the food was eaten!

6. "Anything"

Kende gionsi

  • Refers to any possible thing, open-ended meaning
  • Emphasizes that no specific limit is set
  • Structure:
    • Kende = something
    • Gionsi = any kind/type
  • Example:
    • Ndetere kende gionsi oranche — Bring anything you like.
    • Kende gionsi nabo keratoke aria — Anything could be found there.

7. "Something"

Egento gete

  • Refers to a non-specific item — "something"
  • Used when the speaker has an idea or suspicion of the item but it is not stated
  • Structure:
    • Egente- = class prefix (for things)
    • Gete = one
  • Example:
    • Inaigwete egento gete ase enyomba eria — I heard something in that house.
    • Egento gete ngiatogete ase egesieri — Something was found at the door.
