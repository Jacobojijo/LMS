2.3.4 Noun Plurals

I. Overview — the plural system

The plural system is organised into several major noun classes, each signalled by distinct morphological strategies.

The system includes the following morphological types (as stated in the source material):

  • Invariable forms (singular = plural).
  • Irregular / suppletive forms (plural completely different from singular).
  • Human / agent class with a singular prefix omo- and plural aba-.

The description below treats each class in turn, stating the stated pattern and then listing the provided examples (singular → plural with glosses exactly as given).

II. Class I — -ama Class

Stated pattern: Singular nouns end in various forms; plurals formed by adding -ama.

Morphological impression from the examples: Plural forms show an Ama- sequence in front of or adjoining the base (the examples consistently show Ama- in the plural).

Representative examples (as provided):

  • Eriogi → Amariogi (voice → skins)
  • Risangu → Amasangu (skin → children)
  • Ribuoni → Amarabuoni (Potato → Potatoes)
  • Riruma → Amaruma (dove → doves)
  • Riuga → Amauga (bone → bones)
  • Ritunda → Amatunda (fruit → fruits)
  • Rinani → Amanani (forest → forests)

III. Class II — -eki / -ebi Class

Stated pattern: Plural formed using -eki & -ebi.

Representative examples (as provided):

  • Ekiigo → Ebiigo (monitor lizard → monitor lizards)
  • Ekiarambe → Ebiarambe (wasp → wasps)
  • Ekerogo → Ebirogo (chair → chairs)

IV. Class III — Invariant Plurals (–a)

Stated pattern: No change in plural form (invariant).

Representative examples (as provided):

  • Amache → Amache (water → water)
  • Amabuta → Amabuta (Oil → Oil)
  • Amare → Amare (cloud → clouds)

V. Class IV — Omo → Aba Class (People / Nationalities / Professions)

Stated pattern: People/agent nouns change from the singular prefix omo- to the plural prefix aba-.

Representative examples (as provided):

  • Omoteneneri → Abateneneri (mediator → mediators)
  • Omotomi → Abatomwa (sender → senders)
  • Omwobweni → Ababweni (healer → healers)
  • Omogeni → Abageni (foreign neighbour → foreign neighbours)
  • Omogachi → Abagachi (treasurer → treasures)
  • Omworokia → Abaoroki (teacher → teachers)
  • Omwagachi → Abaagachi (builder → builders)
  • Omogaka → Abagaka (elder → elders)
  • Omorogi → Abarogi (wizard → wizards)
  • Omorusi ebango → Abarusi ebango (tax collector → tax collectors)
  • Omoibi → Abaibi (thief → thieves)
  • Omorebi → Abarebi (drunkard → drunkards)
  • Omotengi → Abatengi (dancer → dancers)
  • Omoiti → Abaiti (killer → killers)
  • Omonyagitari → Abanyagitari (doctor → doctors)
  • Omokungu → Abakungu (woman → women)
  • Omosacha → Abasacha (man → men)
  • Omogomba → Abagomba (barren woman → barren women)

Key formal note: this class is explicitly prefixal alternation: omo- (sg.) → aba- (pl.) across a wide set of human/agental lexis in the provided material.

VI. Class V — -e / -chi Class

Morphological impression from examples: The plural forms show a chi- element (appearing as an initial element Chi- or with Ch- plus base), consistent across examples.

VII. Class VI — -general Class

Stated pattern: "Plurals formed with –different ending."

Representative examples:

  • Egesusu → Ebisusu (rabbit → rabbits)
  • Omotienyi → Emetienyi (moon → moons)
  • Makobiara → Makobiara (mother-in-law → mothers-in-laws)
  • Enchera → Chinchera (road → roads)
  • Ekegwankwa → Ebegwankwa (butterfly → butterflies)
  • Egekondo → Ebikondo (monkey → monkeys)
  • Omwana → Abana (Son in-law →Sons in-law)
  • Ring'ana → Amang'ana (word → words)
  • Egekoroto → Ebikoroto (shoe → shoes)
  • Egioto → Ebioto (frog → frogs)
  • Omorero → Emerero (fire → fires)
  • Oboremo → Amaremo (land → lands)
  • Magokoro → Magokoro (grandmother → grandmothers)
  • Obonyansi → Obonyansi (grass → grasses)

Observation: this class contains a variety of alternations (prefixal alternations such as Ege- → Ebi-, Omo- → Aba-, O- → A-, invariant examples, and other non-uniform patterns). The label "general" in the source suggests it groups heterogeneous plural strategies exemplified above.

Cross-class comparisons & formal observations

Affixal strategies visible in the data:

  • Prefixation (e.g., Ama-, Chi-, Aba-, Ebi-) is frequent in plural formation in the examples.
  • Invariance: some lexemes maintain identical singular and plural forms
  • Suppletion / irregular change: several human/agent forms undergo complete alternation (omo- → aba-; some verb-derived or kinship forms show different plural stems).

Distributional tendencies from the examples:

  • Human/agent nouns are regularly associated with omo- / aba- alternation
  • Animate and inanimate common nouns appear in classes signalled by Ama-, Chi-, Ebi-, etc.

Morphophonological notes based on examples:

  • The same pluralising element (e.g., Ebi-) appears across different stems suggesting class markers may be independent of simple final-segment morphology and instead attach to a lexical stem.
  • Some plural markers appear as initial segments (prefixes) rather than suffixes in the examples despite the original label "-ama": the data show Ama- as an initial element in the plural.
