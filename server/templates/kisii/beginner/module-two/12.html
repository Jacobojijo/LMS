2.4.1 Summary of Nouns (Part 02)

Nouns in Ekegusii: Concrete & Abstract

1. Concrete Nouns

Definition: Tangible, physical objects or beings perceived via the senses.

Categories & Examples:

  • People: Omworokia (teacher), Omosani (friend), Omwana (child)
  • Animals: Esese(dog), Ekebusi (cat), Enchogu (elephant)
  • Places: Egetii/ekegwanda (field), Omochie (city), Esukuru (school)
  • Things: Ekerogo (chair), Egetabu (book), Egari (car)
  • Materials: Chinko (wood), Echuma (steel)

Sample Sentences:

Naroche esese ase omochie. – "I saw the dog in the city."

Aisarasante ae ekerogo naende akaigora egetabu – "She sat on the chair and opened her book."

2. Abstract Nouns

Definition: Intangible concepts, emotions, or qualities not perceived by senses.

Categories & Examples:

  • Emotions: Omogoko (happiness), Okobea (sadness), Obororo (anger), Obwoba (fear)
  • Qualities: Obuya (beauty), Oboitongo (humility), Chinguru (strength), Oboronge (honesty)
  • Ideas: Obwanchani (love), Obosibore (freedom)
  • Conditions: Obochenu (health), Obotaka (poverty), Omorembe (peace)

Sample Sentences:

Omogoko oye nigo ororekanete nyuma y'amang'ana amayia. – "His happiness was evident after the good news."

Oboene nobuya ase kera ekenyoro – "Justice is important in every society."

General Types of Pluralization

  • Invariable forms – singular and plural are identical.
  • Irregular forms – plural is completely different from singular.
  • Human/agent nouns – singular marked by omo-, plural by aba-.

Class I – -ama Class

Pattern: Add -ama to form plurals.

Examples:

  • Eriogi → Amariogi (voice → voices/skins)
  • Risangu → Amasangu (skin → skins/children)
  • Ribuoni → Amaribuoni (potato → potatoes)
  • Ritunda → Amatunda (fruit → fruits)

Class II – -eki / -ebi Class

Pattern: Singular nouns with eki- take ebi- in plural.

Examples:

  • Ekiigo → Ebiigo (monitor lizard → monitor lizards)
  • Ekerogo → Ebirogo (chair → chairs)

Class III – Invariant Plurals (-a)

Pattern: Singular and plural remain the same.

Examples:

  • Amache → Amache (water → water)
  • Amabuta → Amabuta (oil → oil)

Class IV – Omo → Aba Class (Human/Agent Nouns)

Pattern: Omo- (singular) changes to aba- (plural).

Domain: People, professions, nationalities.

Examples:

  • Omotomi → Abatomwa (messenger → messengers)
  • Omogeni → Abageni (guest → guests)
  • Omokungu → Abakungu (woman → women)
  • Omosacha → Abasacha (man → men)
  • Omorogi → Abarogi (wizard → wizards)

Class V – -e / -chi Class

Pattern: Singular nouns take chi- in the plural.

Examples:

  • Enyongo → Chinyongo (pot → pots)
  • Etigere → Chitigere (donkey → donkeys)
  • Ekaramu → Chikaramu (pen → pens)
  • Enyomba → Chinyomba (house → houses)
  • Enyimbo → Chinyimbo (song/rod → songs/rods)

Class VI / VII – General Class (Mixed Patterns)

Pattern: Plural formed with a variety of suffix changes, prefix alternations, or irregular shifts.

Examples:

  • Egesusu → Ebisusu (rabbit → rabbits)
  • Omotienyi → Emetienyi (moon → moons)
  • Enchera → Chinchera (road → roads)
  • Egekoroto → Ebikoroto (shoe → shoes)
  • Omwana → Abana (child/son → children/sons)
  • Ring'ana → Amang'ana (word → words)
  • Oboremo → Amaremo (land → lands)

Some are invariant, e.g. Makobiara → Makobiara (mother-in-law → mothers-in-law).

Key Takeaways

  • Pluralization is class-based, not uniform; each class has distinct morphological markers.
  • Prefixes play a crucial role (omo- → aba-, eki- → ebi-, e-/en- → chi-).
  • Invariant forms exist, especially for mass nouns (amache, amabuta).
  • Human/agent nouns form a large, highly regular class (omo → aba).
  • Irregular/miscellaneous patterns exist, reflecting natural language exceptions.

Noun Construct (Singular & Plural Forms)

What It Is

A base noun form used to:

  • Attach demonstratives (e.g., this/that)
  • Add possessives (covered separately)

Forming the Construct (Singular)

  • Start with plural noun
  • Drop final vowel
  • Egeticha kiaye → Ebiticha biaye-
  • Endagera yaye → chiaye-

Irregular Construct Examples

Meaning Singular Plural Construct
House enyomba chinyomba Chi-
Woman omokungu abakungu Aba-

Plural Construct = Usually Same as Plural

  • Amache aye → Amache aye-
  • Omokungu → Omosacha obwate abakungu abange-

Demonstrative Suffixes (Pointing Words)

Meaning Suffix
This -oyo
That (near) -eke
That (far) -keria
These -ebi
Those (near) -biria
Those (far) -baria

Examples with "enchera" (path)

  • Enchera eyei – this path
  • Enchera eyio – that path (near)
  • Enchera eria – that path (far)
  • Chinchera chiria – these paths
  • Chincher echio – those paths (near)
  • Chinchera chiiria – those paths (far)
