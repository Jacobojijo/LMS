3.3.1 Introduction to Comparatives

1. Introduction to Comparatives

  • Comparatives are used to compare qualities between two or more entities.
  • In Gusii, comparison may involve:
    • Relative comparison (e.g., taller, sweeter)
    • Comparison of equality (e.g., as tall as, as sweet as)
  • These are often formed with:
    • Direct comparative forms
    • The construction "ebwekaine" (similar to)
    • The use of "kaine" (as...as)

2. Adjectives and Their Comparative Forms

The following are selected adjectives and their corresponding comparatives:

Adjective (Positive) Meaning Comparative Meaning
Omotambe tall Omotamba'ke taller
Enyansu sweet Enyansu'ake sweeter
Ekebe bad Ekebe mono worse
Ekiya good Ekiy'ake better
Omong'aini wise/clever Omong'aini ake wiser/cleverer

Example Sentences:

  • Eke negetambe ake kobua keria. → "This is taller than that one."
  • Eke negiansu ake kobua keria. → "This is sweeter than that one."
  • Eke nekebe mono kobua keria. → "This is worse than that one."
  • Eke nekiya kobua keria. → "This is better than that one."
  • Oyo nomong'aini kobua oria. → "This is wiser than that one."

3. Comparison of Equality

Equality in comparison is expressed using:

  • "buna keria" = "similar to" or "as...as"
  • "buna" = used to express "as...as"

Standard Form:

  • "eke [adjective] buna keria" — "this is as [adjective] as that one"
  • Alternatively: "eke [adjective] buna" — same meaning, slightly more flexible form.

Examples:

Construction Translation
Eke negetambe buna keria This is as tall as that one
Eke negiansu buna keria This is as sweet as that one
Eke nekebe buna keria This is as bad as that one
Eke nekiya buna keria This is as good as that one
Oyo nomonyakieni buna oria This is as beautiful as that one
Oyo nomong'aini buna oria This is as wise as that one

4. Use of "buna" in Comparative Sentences

  • The word "buna" functions similarly to the English word "as" in expressions of equality.
  • Used to frame parallel qualities between two entities.

Examples of buna Constructions:

  1. Eke nekebe buna keria → This is as bad as that one
  2. Eke negetambe buna keria → This is as tall as that one
  3. Eke kiagoyire buna eke → This is as weak as that one
  4. Eke negiansu buna keria → This is as sweet as that one
  5. Eke nekiya buna keria → This is as good as that one
  6. Oyo nomong'aini buna oria → This is as wise as that one

5. Morphological Patterns in Comparatives

  • Many Gusii adjectives show a reduction in the syllable when forming the comparative:
    • egetambe → ebitambe
    • egiansu →ebiansu
    • ekebe → ebiya
    • ekiya → ebiya
    • Omong'aini → Abang'aini
  • This pattern is consistent and reflects a grammatical regularity in the formation of comparatives.

6. Cultural Linguistic Note

  • Gusii comparisons often focus not only on physical attributes (height, beauty), but also character and moral qualities (goodness, wisdom, badness).

These reflect the community's emphasis on personality and moral uprightness in communication.
